# 基于知识图谱的RAG知识服务平台

本工作区按论文《基于知识图谱的RAG知识服务平台设计与实现》整理为三个可独立展示的项目，分别对应论文的方法与系统章节。

## 项目映射

- `gai-kgc-lite`
  - 对应第三章 `GAI-KGC`
  - 公开版名称：`GAI-KGC Lite`
  - 侧重知识构建、证据治理、Neo4j 导入与图分析

- `asr-rag-core`
  - 对应第四章 `ASR-RAG`
  - 公开版名称：`ASR-RAG Core`
  - 侧重图文混合检索、知识图谱接口与方法演示

- `industrial-knowledge-service-platform`
  - 对应第五章工业知识服务平台
  - 公开版名称：`Industrial Knowledge Service Platform`
  - 侧重问答工作台、图谱可视化与服务化原型

## 公开版说明

当前目录已按 GitHub 展示场景做了整理：

- 保留了可运行的演示代码与公开样例数据。
- 删除或替换了硬编码密钥、密码、绝对路径等不适合公开的内容。
- 默认只保留轻量样例与演示流程，不包含完整工业语料、全部实验参数、私有提示模板和生产部署配置，因此外部查看者可以运行公开 demo，但不能完整复现实验结果。
